Regence (BlueCross BlueShield) Web Portal

Coding, Content Management, CSS, CSS3, Design, HTML, HTML5, Java, jQuery, PHP, Responsive, SASS/SCSS, Web Design, and Wordpress

Project information

The Regence website was built in 2013 and produced by Cambia.

Regence (Blue Cross and Blue Shield) provides health care plans for Oregon, Utah, and Idaho. The web site provides resources for all visitors and allows members to administer their plans.

Role information

As a UX Developer

I built user-interfaces from images and wireframes. I had ownership of the portal theme, templates, and styleguide. I used Sublime Text to code and Grunt to build. I designed directly in the browser using the Twitter Bootstrap framework and SCSS. I encouraged responsive design, accessibility, and more efficient coding practices.
